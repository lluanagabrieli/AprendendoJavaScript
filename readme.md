Pode-se utilizar tanto no próprio arquivo HTML como também em um arquivo externo.

---------------------------------------------------------------------------------
NO ARQUIVO HTML:
<script> 
    code JavaScript
</script> 

---------------------------------------------------------------------------------
ARQUIVO EXTERNO:
<script scr="arquivo.js">
</script>
---------------------------------------------------------------------------------

Tipos de variáveis:
    -> string - armazenamento de textos, caracteres
    -> number - int/float
    -> boolean - true/false
